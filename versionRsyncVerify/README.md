
## versionRsyncVerify.json

The file `versionRsyncVerify.json` is checked by RsyncVerify at every startup. The [file](https://github.com/rsyncOSX/RsyncVerify/tree/main/versionRsyncVerify/versionRsyncVerify.json) is hosted on Github. The content of URL is `version` and url to new `RsyncVerify.dmg` file. At startup RsyncVerify checks its version number compared to version numbers listed in URL. If there is a match between RsyncVerify version number and URL there is released a newer version.
